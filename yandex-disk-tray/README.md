# Yandex.Disk Tray

Графический индикатор для Яндекс.Диска в системном трее.

## Возможности

- 🟢 Иконка в трее с отображением статуса
- 🔄 Автоматический запуск yandex-disk
- ⏱️ Мониторинг таймаутов
- 🚀 Автозапуск при входе
- 📋 Просмотр логов
- 📂 Быстрое открытие папки
